## 🤖 Agent 1: Meeting Insights AI

**What it does:**
Transforms raw meeting notes into structured 
business intelligence automatically.

**Output:**
- ✅ Meeting summary (2-3 sentences)
- ✅ Action items with owners and deadlines
- ✅ Decisions made
- ✅ Open questions

**Built with:**
Langdock · Prompt Engineering · LLM workflow design

**Use case:**
Eliminates manual post-meeting admin. Paste notes,
get a structured brief in seconds.

## 📊 Agent 2: JD Intelligence Analyser

**What it does:**
Analyses any job description and returns a 
structured recruiter intelligence brief — 
automatically logged to Google Sheets.

**Output:**
- ✅ Hard skills required
- ✅ Soft skills & culture signals
- ✅ Seniority level with reasoning
- ✅ 3 tailored screening questions
- ✅ Red flags worth probing

**Integrations:**
- Google Sheets (auto-appends every analysis)
- Web search (enriches JD context in real time)
- Canvas (formats output as a document)

**Built with:**
Langdock · Google Sheets API · Prompt Engineering

**Note on Google Sheets automation:**
The auto-logging integration is configured and 
connected. On free trial the spreadsheet ID 
pre-fill is locked — the agent asks for 
confirmation once per session before writing.
Full automation available on paid plan.
